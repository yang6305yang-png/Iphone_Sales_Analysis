📌 Project Overview

This project focuses on analyzing iPhone sales data to uncover key insights about pricing, customer preferences, and product performance. The goal is to identify trends in ratings, pricing, and popular models to support better business decisions.

🎯 Objectives

Analyze iPhone product pricing

Identify most and least rated products

Compare different iPhone variants

Extract insights from customer ratings

Visualize trends and patterns

🛠️ Tools & Technologies Used

Python

Pandas

NumPy

Matplotlib

Seaborn

Jupyter Notebook

📂 Dataset Description

The dataset includes:

iPhone model names

Color variants

Storage variants

Price details

Customer ratings

🔍 Key Analysis Performed

Data cleaning and preprocessing

Handling missing values and formatting

Price analysis of different iPhone models

Rating-based comparison

Visualization using graphs and charts

📊 Key Insights

⭐ Most Rated Product: iPhone 8 Plus (Gold, 64GB)

💰 Most Expensive Product: iPhone 12 Pro Max (Silver, 512GB)

💸 Least Expensive Product: iPhone SE (White, 64GB)

📉 Lowest Rated Product: iPhone (Graphite, 128GB)

📈 Visualizations

Bar charts for price comparison

Rating distribution graphs

Product-wise comparison charts

Trend analysis plots
